选择器

选择器

灵活的使用选择器，可以简化操作步骤、实现更多的交互、完成你想不到的效果

选择器分类

  • 元素选择器
  • 类选择器
  • ID选择器
  • 通配符选择器
  • 属性选择器

语法

元素 {样式声明 }

示例

CSS

span {
  background-color: DodgerBlue;
  color: #ffffff;
}

HTML

  <span>这里是由 span 包裹的一些文字.</span>
  <p>这里是由 p 包裹的一些文字.</p>

效果

这里是由 span 包裹的一些文字.

这里是由 p 包裹的一些文字.
